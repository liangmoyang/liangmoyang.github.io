Variable Shadow 变量遮盖 ​

其他称呼：shadow变量

案例一 ​

Go
i := 0
if condition {
    i := 1
    fmt.Println(i) // 1
}
fmt.Println(i) // 0
i := 0
if condition {
    i := 1
    fmt.Println(i) // 1
}
fmt.Println(i) // 0

因为在if的作用于内对变量i使用了:=的重新声明，所以内层的赋值并不会对外层的同名变量产生作用。

案例二 ​

Go
var i int = 0;

if condition {
    i ,err := f1()
    if err != nil{
        return err
    }
} else {
    i ,err := f2()
    if err != nil{
        return err
    }
}

fmt.Println(i)
var i int = 0;

if condition {
    i ,err := f1()
    if err != nil{
        return err
    }
} else {
    i ,err := f2()
    if err != nil{
        return err
    }
}

fmt.Println(i)

在上面这段代码中，先是定义了变量i，然而在if分支中，因为先前没有声明err，所以是使用了:=，这就会导致外层的i永远都是0。 我们可以把err也在外层声明，这样内层就可以使用=：

Go
var i int = 0
var err error

if condition {
    i,err = f1()
}else{
    ···
}
fmt.Println(i)
var i int = 0
var err error

if condition {
    i,err = f1()
}else{
    ···
}
fmt.Println(i)

分析工具 ​

安装工具：go install golang.org/x/tools/go/analysis/passes/shadow/cmd/shadow

使用命令：go vet -vettool=shadow.exe .\main.go
